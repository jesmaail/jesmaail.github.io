Portfolio:

College

For my Computing A-Level I learnt to Program in Visual Basic 6.

The first year Computing project was to create fully functioning program for a fictitious company named Bertie's Buses.
The program was to replace a paper-based booking system, using a database to store Customer information and their purchase quotes.

For the Second year Computing project I chose to stay with Visual Basic 6 and created a Video Game Rental System.
This had very similar functionality to the Bertie's Buses project, however I learnt SQL for communication with the database, and produced a much more polished product.
