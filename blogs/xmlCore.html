Mapping XML in a .Net Core Web API

How to use XML and XML attributes in a ASP.Net Core Web API.
A good introduction for developing a web API in ASP.Net Core can be found here, which uses a trivial model class, and does not cover the extra steps required to make it entirely compatible with XML.

First Steps

Add the following code to the ConfigureServices method in the Startup.cs file in your project. 
services.AddMvc(config =>
{
	config.RespectBrowserAcceptHeader = true;
	config.InputFormatters.Add(new XmlSerializerInputFormatter());
	config.OutputFormatters.Add(new XmlDataContractSerializerOutputFormatter());
});
You'll also want the following using:
using Microsoft.AspNetCore.Mvc.Formatters;
The API can now handle XML as well as JSON formats.

XML Attributes and ASP.Net Core

That will work for straightforward XML without attributes, however we will need to add a few things to our model class(es) if we want to use attributes.

The best way to explain this is to use an example, we want to have the API be able to accept an XML document that looks like: 
<Band>
	<BandMembers>
		<BandMember name="Phil">
			<Instrument>Vocals</Instrument>
			<Availability>Saturdays<\Availability>
		</BandMember>
		<BandMember name="Dave">
			<Instrument>Guitar</Instrument>
			<Availability>Weekends<\Availability>
		</BandMember>
		<BandMember name="James">
			<Instrument>Bass</Instrument>
			<Availability>Everyday<\Availability>
		</BandMember>
		<BandMember name="Chris">
			<Instrument>Drums</Instrument>
			<Availability>Weekends<\Availability>
		</BandMember>
	</BandMembers>
	<Gigs>
		<Gig venue="Motorpoint Arena" date="01/01/2010">
		<Gig venue="Brangwyn Hall" date="10/01/2010">
	</Gigs>
</Band>

So the classes we want for our model should look like: 
public class Band
{
	public List<Member> BandMembers { get; set; }
	public List<Gig> Gigs { get; set; }
}
					                

using System.Xml.Serialization;

[XmlRoot("BandMember")]
public class BandMember
{
	[XmlAttribute(AttributeName = "name")]
	public string Name { get; set; }
	public string Instrument { get; set; }
	public string Availability { get; set; }
}
					                
using System.Xml.Serialization;

[XmlRoot("Gig")]
public class Gig
{
	[XmlAttribute(AttributeName = "venue")]
	public string Venue { get; set; }
	[XmlAttrivute(AttributeName = "date")]
	public string Date { get; set; }
}
